Table Cheats

"tr" Table Row Element

The table row element, "tr" , is used to add rows
to a table before adding table data and table headings.

"td" Table Data Element

The table data element, "td" , can be nested inside
a table row element, "tr" , to add a cell of data to a table.

"thead" Table Data Element

The table head element, "thead" , defines the
headings of table columns encapsulated in table rows.
